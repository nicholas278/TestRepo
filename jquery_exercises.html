Displaying Page Content

Example: show(), hide()

This example uses click events to show() and hide() an image on this page.

Challenge: fade effects

Create three new buttons and assign click events to each. Add the jQuery effect 'fade in', 'fade out', and 'fadeToggle' to show/hide the #profile_image.

  • use fadeIn() to show the image
  • use fadeOut() to hide the image
  • use fadeToggle() to both show/hide the image
Challenge: slide effects

Instead of implementing each with click events on three different buttons, use click events on three anchor tags to test the various slide effects.

  • use slideUp() to hide the image
  • use slideDown() to show the image
  • use slideToggle() to both show/hide the image

Altering Page Content

Example: text()

This example uses click events to change the text() content of the #jquery_message tag.

Challenge: append(), empty(), html()

Create three divs and assign jQuery click events to each. For each of these, you will manipulate the contents of the #jquery_message tag.

  • use append() to append the following text to the tag: "Hello again!"
  • use empty() to clear all content from the tag
  • use html() to change the contents of the #jquery_message tag to include the following html code: <span style='color:red'>jQuery</span> is very powerful!
Challenge: attr()

Add new tags of your choice (button, div, etc). Apply eventa of your choice to each (click, dblclick, etc). Use these events to trigger the actions described below.

  • use attr() to change the src of the #profile_image tag. Change the src from its current value to this: http://bcitcomp.ca/comp1950/lectures/four/pic01.jpg
  • use toggleClass() to swap the CSS class of #jquery_message from .blueclass to .yellowclass

CSS Manipulations

Challenge: css()

Use jQuery CSS functions to alter the #profile_image. As usual, you will need to add new clickable buttons or tags to trigger the jQuery

  • add a solid, black, 2 pixel border to #profile_image
  • change the dimensions of #profile_image to 200px * 200px
  • absolute position #profile_image to the top:10px and left:10px
