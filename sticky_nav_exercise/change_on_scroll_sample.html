COMP1950 Exercise Challenge:
  • Examine the behavior of this page when scrolling away from the top of the page, and scrolling back again to the top. look for the span message created just below this fieldset for information on how jQuery is assessing the scroll
  • Examine the jQuery code embedded in this document. note how it uses scroll events to track the position of the scroll bar.
  • Use jQuery to alter the nav styles when the user scrolls away from the header, apply a fixed position to keep the nav fixed in the top left corner of the browser
  • Bonus challenge: when the user returns to the top of the page, use jQuery to return the nav to original left floated orientation. note: you will need to explicitly override the fixed positioning with static position, as well as reassigning the left floating
  • For further information, investigate scroll and scrollTop
